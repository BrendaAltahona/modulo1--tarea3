Repositorio:
Es el lugar donde se guardan los archivos y cambios de un proyecto. 
Sirve para mantener todo organizado y poder trabajar sin perder 
información importante.

Commit:
Es una forma de guardar los cambios realizados en el proyecto. 
Cada commit funciona como un registro que permite saber qué 
modificaciones se hicieron.

Push:
Es el comando que se usa para enviar los cambios del proyecto 
al repositorio en línea. Así otras personas también pueden ver 
o descargar las actualizaciones.
